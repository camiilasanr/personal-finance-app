# Finora

Aplicación de gestión financiera personal diseñada para ayudar a las personas a controlar sus ingresos, gastos, presupuestos y objetivos financieros mediante una experiencia moderna, intuitiva y multiplataforma.

## Estado del Proyecto

Actualmente el proyecto se encuentra en fase de planificación y diseño.

Durante esta etapa se está trabajando en:

- Definición del alcance del producto.
- Diseño de la arquitectura del sistema.
- Modelado de datos.
- Selección de tecnologías.
- Diseño de experiencia de usuario.
- Planificación del desarrollo.

## Objetivo

Finora tiene como objetivo permitir que cualquier persona pueda:

- Registrar ingresos.
- Registrar gastos.
- Gestionar categorías financieras.
- Administrar presupuestos.
- Gestionar metas de ahorro.
- Visualizar información financiera mediante dashboards interactivos.
- Comprender mejor sus hábitos financieros.
- Tomar decisiones financieras más informadas.

## Gestión de Usuarios

- Registro de usuario.

- Gastos del período.
- Presupuesto utilizado.
- Gráficos de gastos por categoría.

- Resumen financiero general.
